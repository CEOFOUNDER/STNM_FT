# UTM Tracking Guide

Use UTM parameters on every paid diagnostic booking link so Calendly and analytics can show where bookings came from.

## Base Paid Diagnostic Link

https://calendly.com/gilles-bonelli/applied-ai-coaching-session-clone

## Website CTA Format

Use:

`?utm_source=website&utm_medium=cta&utm_campaign=ai_finance_diagnostic&utm_content=[page_or_location]`

Examples:

- Homepage hero:
  `https://calendly.com/gilles-bonelli/applied-ai-coaching-session-clone?utm_source=website&utm_medium=cta&utm_campaign=ai_finance_diagnostic&utm_content=homepage_hero`

- Use case article:
  `https://calendly.com/gilles-bonelli/applied-ai-coaching-session-clone?utm_source=website&utm_medium=cta&utm_campaign=ai_finance_diagnostic&utm_content=use_case_article_cta`

- Process mining article:
  `https://calendly.com/gilles-bonelli/applied-ai-coaching-session-clone?utm_source=website&utm_medium=cta&utm_campaign=ai_finance_diagnostic&utm_content=process_mining_article_cta`

## LinkedIn Format

For direct messages:

`?utm_source=linkedin&utm_medium=dm&utm_campaign=ai_finance_diagnostic&utm_content=[message_type]`

Examples:

- High potential LinkedIn reply:
  `https://calendly.com/gilles-bonelli/applied-ai-coaching-session-clone?utm_source=linkedin&utm_medium=dm&utm_campaign=ai_finance_diagnostic&utm_content=high_potential_reply`

- Possible potential LinkedIn reply:
  `https://calendly.com/gilles-bonelli/applied-ai-coaching-session-clone?utm_source=linkedin&utm_medium=dm&utm_campaign=ai_finance_diagnostic&utm_content=possible_potential_reply`

For LinkedIn posts:

`?utm_source=linkedin&utm_medium=social&utm_campaign=ai_finance_diagnostic&utm_content=[post_theme]`

## Email Format

For direct email follow-up:

`?utm_source=email&utm_medium=direct&utm_campaign=ai_finance_diagnostic&utm_content=[context]`

## Naming Rules

- Use lowercase.
- Use underscores, not spaces.
- Keep `utm_campaign=ai_finance_diagnostic` consistent.
- Use `utm_content` to identify the page, post, reply type or CTA location.
- Do not create a new campaign name unless the offer changes.

## What To Review Weekly

- Which `utm_source` produced bookings?
- Which `utm_content` produced paid diagnostic clicks?
- Which articles generated clicks but no bookings?
- Which LinkedIn replies generated bookings?
- Which pages need stronger qualification copy?
